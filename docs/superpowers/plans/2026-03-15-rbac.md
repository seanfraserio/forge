# RbacManager Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement role-based access control for agent deployments in `@forge-ai/enterprise`, with policy config via constructor and persistent user→role assignments.

**Architecture:** Roles passed as constructor config (not persisted). User assignments stored in `.forge/rbac-assignments.json`. Permission checks follow: lookup assignment → fallback to defaultRole → resolve role → check admin bypass → check permission + environment.

**Tech Stack:** TypeScript (ES2022), node:fs/promises, Vitest

**Spec:** `docs/superpowers/specs/2026-03-15-rbac-design.md`

```typescript
  private async readAssignments(): Promise<Record<string, string>> {
    let raw: string;
    try {
      raw = await readFile(this.assignmentsPath, "utf-8");
    } catch (err: unknown) {
      if (
        err instanceof Error &&
        "code" in err &&
        (err as NodeJS.ErrnoException).code === "ENOENT"
      ) {
        return {};
      }
      throw err;
    }

    try {
      return JSON.parse(raw) as Record<string, string>;
    } catch {
      console.warn("Warning: Corrupted rbac-assignments.json. Treating as empty.");
      return {};
    }
  }

  async assignRole(userId: string, roleName: string): Promise<void> {
    if (!userId || !userId.trim()) {
      throw new Error("userId must be a non-empty string");
    }

    const roleExists = this.policy.roles.some((r) => r.name === roleName);
    if (!roleExists) {
      throw new Error(`Unknown role: "${roleName}". Available roles: ${this.policy.roles.map((r) => r.name).join(", ")}`);
    }

    const assignments = await this.readAssignments();
    assignments[userId] = roleName;

    await mkdir(this.stateDir, { recursive: true, mode: 0o700 });
    await writeFile(this.assignmentsPath, JSON.stringify(assignments, null, 2), {
      encoding: "utf-8",
      mode: 0o600,
    });
  }

  async listRoles(): Promise<Role[]> {
    return this.policy.roles;
  }
```

```typescript
  async checkPermission(userId: string, permission: Permission, environment: string): Promise<boolean> {
    // 1. Look up assigned role
    const assignments = await this.readAssignments();
    const roleName = assignments[userId] ?? this.policy.defaultRole;

    // 2-3. No role found
    if (!roleName) return false;

    // 4-5. Resolve role definition
    const role = this.policy.roles.find((r) => r.name === roleName);
    if (!role) return false;

    // 6. Admin bypasses environment restrictions
    if (role.permissions.includes("admin")) return true;

    // 7. Check permission + environment
    const hasPermission = role.permissions.includes(permission);
    const envAllowed = !role.environments || role.environments.includes(environment);

    return hasPermission && envAllowed;
  }
```
