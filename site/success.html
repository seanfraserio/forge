Quick Start

$npm install @openforge-ai/enterprise
  1. Install the enterprise package alongside your existing Forge CLI. It works as a drop-in addition.
  2. Follow the integration guide — it walks you through audit logging, RBAC, promotions, and secrets step by step.
  3. Set up your team's roles — define who can deploy to dev, staging, and production. The guide includes a role matrix template.
  4. Wire into CI/CD — the guide includes complete GitHub Actions workflows with RBAC checks, audit logging, and secrets injection.
  5. Run the production readiness checklist — security, operational, and go-live checks before your first governed deploy.
